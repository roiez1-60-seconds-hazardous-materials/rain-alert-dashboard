# 📱 מדריך התקנה מהטלפון - צעד אחר צעד

<div dir="rtl">

## 🎯 המטרה
להעלות את מערכת התראות הגשם לאינטרנט **ישירות מהטלפון**, ללא מחשב!

## ⏱️ זמן משוער
15-20 דקות (פעם אחת בלבד)

## 📋 מה שתצטרך
1. טלפון עם אינטרנט ✅
2. כתובת אימייל ✅
3. **זהו!** לא צריך יותר כלום!

---

## 🚀 שלב 1: יצירת חשבון GitHub (5 דקות)

### למה GitHub?
זה המקום שבו הקוד שלך יאוחסן (חינמי לגמרי!)

### איך?

1. **פתח דפדפן** בטלפון (Chrome/Safari)

2. **גלוש ל**: https://github.com/signup

3. **מלא פרטים**:
   - אימייל (שלך)
   - סיסמה (חזקה!)
   - שם משתמש (לדוגמה: `rain-alert-israel`)
   
4. **אמת אימייל**:
   - GitHub ישלח לך מייל
   - פתח אותו ולחץ על הקישור

5. **✅ סיימת!** עכשיו יש לך חשבון GitHub

---

## 📤 שלב 2: יצירת Repository והעלאת קבצים (5 דקות)

### מה זה Repository?
תיקייה שבה כל הקבצים של הפרויקט שלך.

### איך?

1. **בדף הבית של GitHub**, לחץ על כפתור **[+]** בפינה הימנית העליונה

2. **בחר**: "New repository"

3. **מלא**:
   - **Repository name**: `rain-alert-dashboard`
   - **Description** (אופציונלי): "מערכת התראות גשם בזמן אמת"
   - **סמן**: ✅ Public (שהכל יוכלו לראות)
   - **לא צריך** לסמן Add README
   
4. **לחץ**: "Create repository" (כפתור ירוק)

5. **עכשיו תעלה קבצים**:
   - לחץ "uploading an existing file"
   - אני אתן לך כעת קובץ ZIP - **הורד אותו** 📥
   - **גרור את כל הקבצים** או לחץ "choose your files"
   - בחר את כל הקבצים מהתיקייה שהורדת
   - למטה, כתוב בתיבה: `Initial commit - מערכת התראות גשם`
   - לחץ "Commit changes" (כפתור ירוק)

6. **✅ מעולה!** הקבצים שלך ב-GitHub

---

## 🌐 שלב 3: חיבור ל-Vercel (5 דקות)

### למה Vercel?
זה המקום שבו האתר שלך ירוץ בפועל (חינמי וחזק!)

### איך?

1. **גלוש ל**: https://vercel.com/signup

2. **לחץ על**: "Continue with GitHub"

3. **אשר**: Vercel יבקש גישה ל-GitHub שלך - **תן אישור**

4. **✅ עכשיו אתה מחובר!**

---

## 🚀 שלב 4: יבוא הפרויקט (5 דקות)

### זה השלב האחרון! 🎉

1. **בדף הבית של Vercel**, לחץ על: **"Add New..."**

2. **בחר**: "Project"

3. **אתה תראה רשימה** של ה-Repositories שלך מ-GitHub

4. **מצא את** `rain-alert-dashboard`

5. **לחץ על**: "Import" ליד שם הפרויקט

6. **הגדרות** (חשוב!):
   
   בחלון שנפתח, **שנה רק את זה**:
   
   - **Framework Preset**: שנה ל-"Other"
   - **Root Directory**: השאר `.`
   - **Build Command**: **מחק את מה שיש ותשאיר ריק!**
   - **Output Directory**: כתוב `frontend`
   - **Install Command**: השאר את ברירת המחדל

7. **לחץ**: "Deploy" (כפתור כחול גדול)

8. **המתן 1-2 דקות** ⏳
   - Vercel יבנה את האתר
   - אתה תראה אנימציה של בנייה

9. **✅ זהו!!! הצלחת! 🎉🎊**

---

## 🎯 השלב האחרון - כתובת האתר שלך!

כשהפריסה הושלמה, תראה:

### 🌐 קישור לאתר שלך!

זה ייראה משהו כמו:
```
https://rain-alert-dashboard-abc123.vercel.app
```

**זהו! האתר שלך חי! 🚀**

## ✨ מה עכשיו?

### 1. שמור את הקישור
שמור את כתובת האתר שלך כסימניה בטלפון

### 2. שתף!
שלח את הקישור לחברים/משפחה

### 3. הפעל עדכונים אוטומטיים (אופציונלי)

אם תרצה שהמערכת תתעדכן אוטומטית כל 10 דקות:

1. **ב-GitHub**, עבור ל-Repository שלך
2. לחץ על **Settings** (למעלה)
3. בצד שמאל, לחץ **Actions** → **General**
4. **סמן**: "Allow all actions and reusable workflows"
5. **לחץ**: Save

**זהו!** עכשיו GitHub ירוץ את הסקריפט כל 10 דקות ויעדכן את נתוני המכ"ם.

## 🎨 התאמה אישית

רוצה לשנות משהו באתר?

1. **ב-GitHub**, לחץ על הקובץ שרוצה לערוך
2. לחץ על **✏️ Edit** (עיפרון)
3. ערוך מה שרוצה
4. למטה, לחץ **Commit changes**
5. **Vercel יעדכן אוטומטית!** תוך 1-2 דקות

---

## 🆘 עזרה! משהו לא עובד

### האתר לא נפתח?
- המתן 2-3 דקות - לפעמים לוקח זמן
- רענן את הדף (החלק למטה בטלפון)

### הקבצים לא עלו ל-GitHub?
- ודא שסיימת את תהליך העלאה
- נסה שוב - זה לפעמים קורה

### Vercel אומר "Error"?
- בדוק שה-Output Directory הוא `frontend`
- בדוק שה-Build Command **ריק לגמרי**
