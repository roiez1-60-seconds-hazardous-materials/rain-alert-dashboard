## 🎯 מה אתה עומד לעשות?

להעלות מערכת התראות גשם מקצועית לאינטרנט - **ישירות מהטלפון!**

---

## 🚀 שלושה שלבים פשוטים

### 1️⃣ GitHub (2 דקות)

```
1. גלוש ל: github.com/signup
2. הירשם (אימייל + סיסמה)
3. לחץ [+] → New repository
4. שם: rain-alert-dashboard
5. לחץ: Create repository
6. העלה את הקבצים מה-ZIP
```

### 2️⃣ Vercel (2 דקות)

```
1. גלוש ל: vercel.com/signup
2. לחץ: Continue with GitHub
3. לחץ: Add New... → Project
4. בחר: rain-alert-dashboard
5. לחץ: Import
```

### 3️⃣ הגדרות (1 דקה)

```
Output Directory: frontend
Build Command: [השאר ריק!]
לחץ: Deploy
```
